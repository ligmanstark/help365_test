Продуктовый каталог

Раздел Продуктовый каталог позволяет создать иерархический каталог продуктов компании с характеристиками, для заполнения которых можно настраивать правила.

Здесь вы можете:

  • создавать иерархию продуктов по категориям;
  • группировать характеристики по их тематике на разных вкладках карточки продукта. Это упрощает создание продукта с большим списком данных;
  • добавлять характеристики как с постоянными значениями, так и настраиваемыми;
  • задавать для характеристик диапазоны значений или зависимость от других характеристик.

После создания каталога продуктов сотрудник выбирает нужный продукт в задаче, полученной в ходе процесса, или в карточке элемента приложения, например, в лиде или сделке. При этом он может заполнить конфигурируемые характеристики продукта определёнными значениями только в соответствии с заданными настройками.

Чтобы выбрать, отобразить и заполнить данные продукта на форме приложения или задачи в бизнес-процессе, используется виджет Выбор продукта.

Начало внимание

Раздел Продуктовый каталог доступен при активации одного из платных решений CRM, в состав которого он входит. Использовать его смогут только пользователи с лицензией решения.

Конец внимание 

Раздел Продуктовый каталог позволяет работать с двумя типами продуктов:

  • Простой — содержит только постоянные значения характеристик, установленные при создании продукта в каталоге. Сотрудник выбирает готовый продукт с конкретными значениями в карточке лида или сделки. Например, банковский продукт Дебетовая карта имеет характеристику Гражданство с предустановленным значением РФ. Другие характеристики этого продукта также будут иметь конкретные значения;
  • Конфигурируемый — при создании в каталоге такого продукта можно определить правила заполнения его характеристик. Сотрудник выбирает продукт в карточке лида или сделки и указывает для конфигурируемой характеристики конкретное значение из диапазона или списка допустимых. Например, в продукте Ипотека Молодая семья для характеристики Возраст можно указать значение от 18 до 35 лет, а для характеристики Ставка в % допустимое значение будет разным в зависимости от суммы и срока ипотеки.

Начало примечание

Примечание

Раздел Продуктовый каталог расширяет возможности работы с продуктами компании по сравнению со справочником Продукты из раздела СРМ. Чтобы сотрудники могли использовать в работе с клиентами конфигурируемые продукты, используйте раздел Продуктовый каталог вместо справочника Продукты раздела СРМ.

Конец примечание

Состав раздела «Продуктовый каталог»

В разделе Продуктовый каталог ответственные за создание продуктов пользователи работают со следующими приложениями:

  • Каталог продуктов — хранит перечень простых и конфигурируемых продуктов. Здесь можно перейти к созданию продукта;
  • Группы продуктов — справочник для создания групп продуктов. Каждая группа содержит свой перечень характеристик;
  • Категории продуктов — справочник для настройки иерархии продуктов в каталоге;
  • Характеристики — справочник для создания характеристик продукта;
  • Категории характеристик — справочник для распределения характеристик по группам исходя из их тематики. Группы отобразятся на разных вкладках в карточке продукта.

Также раздел содержит служебные компоненты для хранения продуктов, созданных с помощью виджета Выбор продукта, например в лидах и сделках. Характеристики такого продукта заполнены конкретными значениями. Пользователи не работают с этими компонентами, и администратор системы может скрыть их в левом меню раздела:

  • Продукты — служебный контракт, который используется в настройках виджета Выбор продукта. Источниками контракта являются приложения, размещённые под разделителем Приложения по группам продуктов. На странице контракта отобразятся все элементы этих приложений;
  • Приложения по группам продуктов — под этим разделителем автоматически создаётся служебное приложение, когда пользователи создают элемент в справочнике Группы продуктов. В служебное приложение с названием соответствующей группы сохранится продукт, выбранный с помощью виджета Выбор продукта, например, в карточке лида.

Настроить работу с разделом «Продуктовый каталог»

Чтобы пользователи могли добавлять продукт, например, на карточку лида и сделки, нужно:

  1. Заполнить справочники в разделе Продуктовый каталог. После этого сотрудники могут создавать продукты в приложении Каталог продуктов;
  2. Разместить виджет Выбор продукта на формы приложений Лиды и Сделки. С помощью виджета сотрудник может выбрать и заполнить значениями конфигурируемые характеристики продукта, в котором заинтересован клиент.

Также администратор системы может настроить:

  • набор доступных действий в карточке элемента приложения Каталог продуктов. По умолчанию продукт можно редактировать. Также можно добавить кнопки для удаления элемента, запуска процесса и другие;
  • смену статуса продукта. По умолчанию для продукта заданы статусы Черновик, Актуальный и Неактуальный.
