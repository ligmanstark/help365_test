Отчёт «Мои поступления»

На странице сотрудники, включённые в группы продаж, смогут посмотреть отчёты по плановым и фактическим платежам, за которые они являются ответственными, а также отследить выполнение личных целей продаж.

Страница содержит три отчёта, расположенных в отдельных вкладках:

  • Мои показатели;
  • Поступления;
  • Цели продаж.

Настройка страницы

За отображение каждого отчёта отвечают отдельные виджеты, размещённые на вкладках. Администраторы системы могут настраивать страницу с отчётами:

  • изменять настройки виджетов;
  • менять расположение виджетов на вкладке;
  • удалять графики, статистика по которым не используется в компании;
  • добавлять на вкладку дополнительные виджеты системы.

С виджетами, используемыми на странице Мои поступления, можно ознакомиться в статье «Виджеты отчётов в разделе „Поступления“». Подробнее о настройке страницы в дизайнере интерфейсов читайте в статье «Добавить виджет на страницу».

Чтение отчётов

Мои показатели

Сравнение поступлений, за которые ответственен текущий сотрудник, по суммам цели, плана и факта за определённый период. На основе данных высчитываются проценты выполнения поставленной цели продаж.

Отчёт строится по данным из приложений Реестр поступлений и Цели продаж. Чтобы ознакомиться со статистикой:

  1. Задайте период, данные по которому нужно отобразить.
  2. Укажите параметры поиска в полях для фильтрации: Группа продаж, Контрагент, Статья дохода, ЦФО и Статус платежа.
  3. Нажмите кнопку Построить отчет.

По выбранным данным сформируются три отчёта:

  • Вверху страницы с помощью виджета Мои поступления отображаются суммы: цель, план и факт. На основе сравнения этих показателей высчитываются и отображаются проценты выполнения плана и текущей цели. В зависимости от прогресса в выполнении цели, показатель процента выделяется зелёным или красным цветом.

Справа отображается текущий статус цели и оставшаяся до её выполнения сумма дохода.

  • Статьи дохода — в виджете отображается распределение всех поступлений по определённым статьям дохода. В поле План/Факт над графиком вы можете указать, какую статистику отображать: по запланированным или поступившим платежам.
  • Плановые и фактические поступления — виджет показывает соотношение запланированных платежей и поступивших оплат в виде диаграммы. Статистику по всему заданному периоду можно отобразить по дням, неделям или месяцам. Для этого в поле Отображение выберите нужную опцию.

Поступления

Список плановых и фактических поступлений, за которые ответственен текущий сотрудник. Отчёт строится по данным из приложения Реестр поступлений и отображается в виде таблицы с помощью виджета Поступления. В отчёте можно посмотреть основание для поступления, планируемую и фактическую даты оплаты, сумму и статус платежа, а также наличие счета.

Чтобы отсортировать данные, вверху страницы заполните поля План. дата оплаты и Факт. дата оплаты. Также вы можете отфильтровать данные в отчёте по статусу и вероятности. Для этого нажмите кнопку  и заполните поля в открывшемся окне.

Нажмите Построить отчёт.

По плановому поступлению можно создать фактическое. Для этого нажмите на плюс в столбце Факт. сумма. Кроме того, можно отредактировать плановую дату поступления и его вероятность. Для этого нажмите на значок карандаша рядом с нужным значением.

Цели продаж

Статистика поступлений по суммам цели, плана и факта для определённого сотрудника. На основании сравнения плановых и фактических показателей с целевым значением вычисляется процент выполнения поставленных целей.  

Например, сотрудник может посмотреть, на сколько процентов его группа продаж выполнила цель за прошедший год.

Отчёт строится с помощью виджета Распределение целей продаж по данным из приложений Реестр поступлений и Цели продаж и отображается в виде таблицы.

Данные можно сортировать, например, сотрудник сможет отследить свой прогресс в продажах по месяцам прошлого года. Для этого выберите период. Если вы состоите в нескольких группах продаж, вы можете просмотреть показатели каждой группы по отдельности, указав её название. Нажмите Построить отчёт.

В таблице отобразятся поставленные цели по выручке для компании, групп продаж и текущего сотрудника, а также суммы доходов по запланированным и фактическим поступлениям.

На основе сравнения этих показателей высчитываются проценты выполнения всех поставленных целей.

Пользователи с доступом на создание и редактирование элементов приложения Цели продаж могут добавлять и изменять цели из таблицы. При этом запись создаётся или редактируется в приложении Цели продаж.

Чтобы создать или отредактировать цель продаж, в столбце Цель выберите ячейку и нажмите на значок . В появившемся поле введите или измените сумму и нажмите . Для отмены действия нажмите .
