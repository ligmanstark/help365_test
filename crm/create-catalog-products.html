Создать каталог продуктов

Вы можете перейти к наполнению каталога продуктов, если заполнены справочники раздела Продуктовый каталог. Созданный продукт станет доступен для выбора в элементе приложения, где используется виджет Выбор продукта, например в карточке лида или сделки.

Начало внимание

Работа с каталогом продуктов доступна только для пользователей с лицензией платного решения CRM, в состав которого включён раздел Продуктовый каталог.

Конец внимание 

В приложении Каталог продуктов вы можете создавать продукты двух типов:

  • Простой — содержит только постоянные значения характеристик. Сотрудник выбирает готовый продукт с заданными значениями, например, в карточке лида или сделки. Так, банковский продукт Дебетовая карта является простым, все его характеристики имеют определённые значения. Например, для характеристики Гражданство задано значение РФ;
  • Конфигурируемый — для такого продукта можно определить правила заполнения его характеристик. Сотрудник выбирает продукт в карточке лида или сделки и указывает для конфигурируемой характеристики конкретное значение из допустимых. Например, в продукте Ипотека Молодая семья для характеристики Возраст можно указать значение от 18 до 35 лет, а для характеристики Ставка в % допустимое значение будет разным в зависимости от суммы и срока ипотеки.

Рассмотрим, как создать каждый тип продукта на примере банковских продуктов Дебетовая карта и Ипотека Молодая семья.

Добавить продукт

Чтобы добавить продукт:

  1. На странице приложения Каталог продуктов в правом верхнем углу нажмите + Продукт. Откроется форма создания, которая состоит из двух вкладок: Основное и Настройки характеристик.
  2. На вкладке Основное заполните следующие поля:

    • Название* — введите название продукта;
    • ID продукта — укажите идентификатор продукта в ELMA365;
    • Внешний ID продукта — заполните это поле, если используется импорт продуктов из внешней системы или экспорт, например, с помощью методов АРI;
    • Начало действия*, Окончание действия* — укажите период, когда созданный продукт актуален для использования. Например, вы можете создать черновик продукта. Применять его в работе с клиентами и выбирать на карточках лидов и сделок можно, начиная с даты начала действия и до окончания срока действия. Далее продукт становится устаревшим. Администратор системы может настроить смену статуса продукта в соответствии с указанными сроками;
    • Категории продуктов* — выберите элемент справочника Категории продуктов. В левом меню приложения Каталог продуктов новый элемент отобразится в выбранной папке иерархии всех продуктов, а также в её родительских папках;
    • Группа продуктов* — выберите элемент справочника Группы продуктов. После этого на вкладке Настройки характеристик отобразится список характеристик, заданный для выбранной группы;
    • Ответственный департамент — укажите, кто отслеживает актуальность продукта. Для выбора доступны пользователь, группа или элемент оргструктуры. Администратор системы может ограничить доступ к редактированию карточки продукта только для ответственного;
    • Тип продукта — выберите вариант Простой или Конфигурируемый.
  1. Перейдите на вкладку Настройки характеристик и заполните её в зависимости от выбранного типа продукта:
    • укажите значения для характеристик простого продукта;
    • или настройте характеристики конфигурируемого продукта.
  1. Сохраните продукт.

Настройки характеристик простого продукта

Рассмотрим на примере простого продукта Дебетовая карта, как заполнить вкладку Настройки характеристик. Её внешний вид зависит от параметров, заданных на вкладке Основное:

  • список характеристик определяется выбранной группой продуктов Дебетовые карты;
  • для каждого поля есть возможность указать только конкретное значение, т. к. выбран тип продукта Простой.

Все поля для заполнения отобразятся на двух вкладках Характеристики клиента и Характеристики продукта в соответствии с настроенными категориями для любых продуктов.

Вы можете создать индивидуальный набор характеристик для нового продукта. Для этого на любой вкладке добавьте поле с помощью кнопки + Характеристика или удалите уже существующее.

Заполните значения характеристик продукта Дебетовая карта:

  1. На вкладке Характеристики клиента в поле Гражданство укажите значение РФ.

  1. Заполните значения полей на вкладке Характеристики продукта:

  1. Сохраните настройки. После этого продукт Дебетовая карта добавится в приложение Каталог продуктов.

Настройки характеристик конфигурируемого продукта

Рассмотрим на примере конфигурируемого продукта Ипотека Молодая семья, как заполнить вкладку Настройки характеристик.

Предварительно на вкладке Основное для этого продукта выберите группу продуктов Ипотека и установите тип продукта Конфигурируемый. Эти данные определят внешний вид вкладки Настройки характеристик:

  • список характеристик зависит от настроек, заданных в группе продуктов Ипотека;
  • для каждого поля появится возможность настроить диапазон значений и правила, т. к. выбран тип продукта Конфигурируемый.

Заранее определите, для каких характеристик продукта Ипотека Молодая семья нужно:

  • задать постоянные значения, как в простом продукте;
  • установить выбор значений;
  • настроить правило, то есть зависимость от других характеристик.

Если нужно, вы можете добавить поля с помощью кнопки + Характеристика или удалить уже существующие.

В нашем примере выполните настройки характеристик последовательно на двух вкладках, начиная с вкладки Характеристики клиента.

Шаг 1. Задать постоянные значения

Настройте постоянные значения в полях:

  • Гражданство — установите значение РФ;
  • Наличие кредитной истории — установите значение Нет.

Шаг 2. Установить выбор значений для характеристики

В других полях на вкладке Характеристики клиента установите диапазоны значений, чтобы затем сотрудник мог указать в них только допустимое значение. Для этого:

  1. Включите опцию Конфигурируемая для полей Возраст, Постоянный доход, Стаж на последнем месте работы.
  2. Выберите одно из полей и нажмите Настроить.
  3. В открывшемся окне на вкладке Выбор значений укажите нужный диапазон и сохраните настройки поля.

Далее для полей на вкладке Характеристики продукта также задайте постоянные значения и выбор допустимых значений.

Шаг 3. Настроить правило для характеристики

Если в вашем продукте есть характеристики, значения которых зависят от других характеристик, для них можно настроить правило. Так, в продукте Ипотека Молодая семья есть характеристика Ставка в %, значение которой определяется суммой и сроком ипотеки:

  • если сумма ипотеки составляет от 500 000 до 10 000 000, а срок 120 дней и более, то ставка устанавливается в размере 19%;
  • в других случаях ставка составит 18%.

Настройте указанное выше правило для поля Ставка в %. Для этого:

  1. В поле Ставка в % включите опцию Конфигурируемая и нажмите Настроить.
  2. В открывшемся окне выберите опцию Настройка правил. Отобразится блок Правило 1.
  3. Нажмите + Добавить условие и заполните параметры условия: Поле, Операция, Значение, Связка. Вы можете добавить несколько условий.
  4. В поле Результат установите диапазон значений для ставки, которые сотрудник сможет указать при заполнении данных продукта в карточке лида или сделки, если выполнится заданное условие. В нашем примере введём значение 19. Оно отобразится в подсказке для поля Ставка в %, и сотрудник сможет указать только это значение, иначе отобразится ошибка.
  5. В блоке Иначе укажите, каким диапазоном значений можно будет заполнить поле Ставка в %, если заданное правило не выполнится. В нашем примере это значение 18

  1. Сохраните настроенные правила, а затем сохраните созданный продукт.

После этого продукт Ипотека Молодая семья добавится в приложение Каталог продуктов.

После создания простого продукта Дебетовая карта и конфигурируемого продукта Ипотека Молодая семья сотрудник может выбирать их в виджете Выбор продукта. При этом заполнить конфигурируемые характеристики можно только допустимыми значениями. Подробнее читайте в статье «Выбрать продукт в карточке элемента приложения».
