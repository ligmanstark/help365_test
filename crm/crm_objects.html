Справочники раздела «CRM»

В справочниках хранятся данные, которые нужны для работы в разделе CRM. Для удобства вся информация структурирована в виде списков и таблиц. Записи в справочниках добавляются пользователями ELMA365, которым предоставлены права доступа на создание элементов.

начало внимание

Справочники доступны при активации одного из платных решений CRM, в которые включена работа с лидами и сделками. Использовать их смогут только пользователи с лицензией решения.

конец внимание

В разделе Справочники вы можете:

  • создать список продуктов, которые предоставляет ваша компания. Продукты связаны с приложением Лиды. При добавлении лида вы можете сразу указать, в каком продукте заинтересован потенциальный клиент. Вместо справочника Продукты можно использовать раздел Продуктовый каталог, который позволяет создавать как простые, так и конфигурируемые продукты;
  • указать Отрасли и Сегменты бизнеса контрагентов. Это поможет отследить, среди каких компаний ваша продукция пользуется наибольшим спросом. Отрасли и сегменты связаны с приложением Компании. При создании элементов в связанном приложении вы можете указать необходимое значение из справочника;
  • добавить Маркетинговые мероприятия. С карточки мероприятия можно назначать задачи коллегам и обсуждать необходимые вопросы в ленте событий;
  • указать Источники лидов. Это могут быть, например, входящие звонки, веб-форма на вашем сайте, email‑рассылка и т. д. В дальнейшем вы сможете проанализировать, какой способ продвижения наиболее эффективен для вашей компании;
  • создать Группы продаж, которые помогут распределить сотрудников, ответственных за продажи, по различным категориям, таким как филиалы вашей компании. В отличие от других справочников, доступ к этому приложению есть только у администраторов системы.

Когда администратор добавляет группу продаж, в настройках этого справочника на странице Группы и роли создаётся соответствующая группа пользователей. Эту группу можно использовать для гибкой настройки прав доступа к приложениям CRM, а также при создании индивидуальных и групповых целей продаж в разделе Поступления.

Состав группы можно изменить, отредактировав соответствующую запись в справочнике Группы продаж.

Поскольку Справочники являются приложениями, для них доступны стандартные настройки: вы можете настроить бизнес-процессы, пользовательские формы, ограничивать доступ к данным, добавлять кнопки действия и т. д. Чтобы открыть меню настроек, нажмите на значок шестерёнки рядом с названием нужного справочника.

По умолчанию записи справочников отображаются в виде таблицы. Вы можете изменить способ отображения на канбан-доску при помощи переключателя вверху страницы .

Как добавить запись в справочники

Способы добавления записей одинаковы для всех справочников. Вы можете создавать их вручную и автоматически с помощью API.

Подробнее о том, как добавить элемент приложения при помощи API читайте в справке по публичному API ELMA365.

Добавить запись вручную можно несколькими способами:

  1. В разделе CRM перейдите на страницу нужного вам справочника и нажмите кнопку +Название элемента, например, +Отрасль. 

  1. На карточке создания или редактирования элемента связанного приложения нажмите на кнопку +Создать или значок . Например, при добавлении Компании вы можете создать отрасль, к которой она принадлежит.  

При добавлении нового элемента необходимо заполнить обязательное поле Название. Всю остальную информацию вы можете указать позднее на карточке элемента.
