Работа с поставщиками

Бизнес-решение Закупки содержит раздел SRM, в котором можно организовать взаимодействие с поставщиками:

  • проводить проверки и аккредитации;
  • формировать рейтинг.

Рассмотрим, как строится работа с поставщиками в решении.

Проверка и аккредитация поставщика

Проведение проверок и аккредитаций в решении состоит из следующих этапов:

  1. По каждому контрагенту, добавленному в приложение Поставщики, можно запустить процесс проверки поставщика. В ходе него ответственный анализирует информацию о компании и проводит оценку рисков. Если в системе настроена интеграция с сервисами СПАРК, Контур.Призма и Контур.Фокус, сотрудник видит результаты внешних проверок. Также он может запрашивать дополнительную информацию у поставщика. Контрагент, прошедший проверку, получает статус Активный. Ответственный присваивает ему первоначальный рейтинг.
  2. После этого поставщиков можно фильтровать по статусу, чтобы работать только с проверенными компаниями, например, при выборе исполнителей для заказов.
  3. Проверенного поставщика можно аккредитовать, подтвердив, что он подходит для исполнения заказов по определённому направлению закупки. Эксперт анализирует данные компании, запрашивает дополнительную информацию и принимает решение, устанавливая для контрагента новый рейтинг. При проведении закупочной процедуры аккредитованным поставщикам по направлению закупки можно автоматически высылать приглашения для участия.

Проверки и аккредитации имеют ограниченный срок, который задаётся ответственным при их проведении. С определённой периодичностью в системе запускается процесс, с помощью которого находятся все проверки и аккредитации с истекающим сроком действия. По ним ставятся задачи повторно провести процедуры. Частоту запуска процесса определяет администратор системы при первичной настройке решения Закупки.

Рейтинг поставщиков

Рейтинг поставщиков изменяется в процессе взаимодействия с ними и помогает выбирать для заказов исполнителей, которые лучше всего соблюдают установленные договорённости.

Каждому контрагенту присваивается первоначальный рейтинг по итогам проверки, затем эксперт может изменить его при аккредитации. После этого значение рейтинга можно повысить или понизить в ходе работы с ним:

  • вручную в приложении Изменения рейтинга поставщика — сотрудник может изменить рейтинг на любом этапе работы;
  • в ходе бизнес-процесса с помощью специальных блоков — администратор системы может разместить блоки изменения рейтинга на схемах процессов, связанных с обработкой закупок и заказов. Тогда после определённого этапа процесса рейтинг будет меняться одним из способов:
    • автоматически — на основе значения, заданного при настройке бизнес-процесса;
    • вручную — пользователю назначается задача оценить взаимодействие с поставщиком.

Все изменения рейтинга поставщика отображаются в его карточке в виде графика. В нём можно увидеть историю оценок, а также посмотреть их причины и комментарии.
