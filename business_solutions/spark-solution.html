Интеграция СПАРК

Бизнес-решение Интеграция СПАРК позволяет получать данные по юридическим и физическим лицам, а также их деятельности из базы сервиса проверки контрагентов СПАРК. На основе уникальных показателей сервиса вы сможете выполнить экспресс-анализ клиента, оценить его благонадёжность и принять решение по дальнейшему сотрудничеству.

Отправку запросов в СПАРК можно автоматизировать с помощью бизнес-процессов и отображать полученные данные в удобном виде в интерфейсе ELMA365, например, в карточке контрагента. Также можно вручную отправить нужный запрос, используя преднастроенные процессы в приложениях решения.

Установка платного решения Интеграция СПАРК предоставляет следующие возможности:

  • простое создание карточки компании — укажите название или ИНН контрагента для поиска по базе сервиса СПАРК и загрузки остальных реквизитов;
  • расширенная справка по контрагенту — запрашивайте контактные данные, адреса, реквизиты, а также информацию об их изменении;
  • статус и индексы риска компании — наглядно отобразите показатели в интерфейсе ELMA365;
  • отчёт СПАРК-риски — сформируйте файл формата .pdf с полными данными по компании;
  • запросы для оценки благонадёжности контрагента — получите данные об актуальных и бывших совладельцах компании, дочерних компаниях, а также санкционных рисках и факторах риска компании или физического лица;
  • мониторинг изменений по компании — подпишитесь на контрагента и получайте данные о всех изменениях;
  • конструктор санкционных списков — настройте зоны риска для оценки компании или физического лица на основе актуальных данных из санкционных списков СПАРК.

Состав бизнес-решения

После загрузки бизнес-решения Интеграция СПАРК в систему добавятся следующие компоненты:

  • раздел СПАРК — содержит:
    • преднастроенные приложения для запросов в сервис СПАРК. Вы можете получать данные о контрагенте, а также включать и отключать мониторинг изменений по нему. Запрос создаётся при добавлении элемента в приложение вручную или автоматически в ходе бизнес-процесса. Результаты запроса и информация отображаются в карточке элемента;
    • справочники для настройки зон риска и работы с санкционными списками СПАРК. Создайте правила, по которым контрагент будет отнесён к той или иной зоне риска после получения данных из СПАРК. Результат можно показать в карточке элемента приложения с помощью виджета;
  • модуль Интеграция СПАРК — используется для настройки интеграции с аккаунтом СПАРК. Содержит специальные графические элементы для моделирования бизнес-процессов, а также преднастроенные виджеты поиска контрагента и отображения его статуса и индексов риска, определения зоны риска для контрагента и поиск адреса. С их помощью проверка клиентов по базе сервиса и поиск данных внедряется в пользовательские процессы компании, а результаты добавляются на форму любого приложения;
  • Приложение для БП интеграции СПАРК в разделе CRM — служебное приложение с преднастроенными бизнес-процессами, использующими графические элементы модуля Интеграция СПАРК. Процессы запускаются при создании запросов в сервис вручную.

Установка решения и подключение модуля

Бизнес-решение Интеграция СПАРК импортируется из каталога ELMA365 Store. Чтобы загрузить решение и настроить интеграцию, выполните следующие действия:

  1. На главной странице, в левом меню нажмите на значок . В открывшемся окне Настройка Разделов и Решений перейдите на вкладку Решения, нажмите кнопку + Добавить решение и выберите Скачать.
  2. В каталоге ELMA365 Store выберите решение Интеграция СПАРК и установите его, нажав Попробовать бесплатно.
  3. Поскольку бизнес-решение является платным, его необходимо активировать. Для этого перейдите в раздел Администрирование > Управление лицензиями и напротив названия решения нажмите кнопку Активировать.
  4. После установки решения включите и настройте модуль Интеграция СПАРК. Для этого перейдите в раздел Администрирование > Модули > Интеграция СПАРК. На странице настроек заполните необходимые поля.

  • URL*, Логин*, Пароль* — данные вашего аккаунта в СПАРК, используемые для подключения к API сервиса. После заполнения полей проверьте корректность установленного подключения;
  • Сервис подсказок — URL для подключения дополнительного сервиса СПАРК. В интеграции с ELMA365 сервис подсказок используется для идентификации контрагента по его реквизитам, а также полного адреса по его отдельным данным. Для этого на формы приложений можно добавить виджет Поиск компании или Поиск адреса. Например, в процессе создания карточки компании вы сможете указать её ИНН, а остальные данные загрузятся из базы СПАРК автоматически.
  1. Сохраните настройки.

Первичные настройки завершены. Подробнее о решении читайте в статьях:

  • «Типы запросов в сервис СПАРК» — возможности интеграции с сервисом СПАРК;
  • «Отправка запросов в сервис из раздела „СПАРК”» — работа с разделом и создание запросов вручную;
  • «Конструктор санкционных списков» — настройка зон риска для отнесения контрагента к одной из них на основе данных из санкционных списков СПАРК;
  • «Моделирование процесса с блоками решения „Интеграция СПАРК“», «Входные и Выходные атрибуты блоков решения „Интеграция СПАРК“» — внедрение интеграции в пользовательские процессы компании;
  • «Настройки отображения данных СПАРК на форме приложения» — изменение формы приложения для внедрения интеграции;
  • «Виджеты решения «Интеграция СПАРК» и их настройка» — размещение доступных виджетов на формах приложения для наглядного отображения данных СПАРК.
