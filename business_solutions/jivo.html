Jivo

Jivo — это размещённый на сайте онлайн-чат для общения с клиентами. Благодаря модулю интеграции с Jivo, вы можете передавать в систему ELMA365 контактную информацию об обратившихся на сайте клиентах, создавать новые лиды или сделки, а также сохранять историю переписки с ними.

Перед подключением модуля интеграции убедитесь, что ваш аккаунт в Jivo активен, а на сайте размещён виджет с чатом.

Начало внимание 

Настраивать модули могут только пользователи, входящие в группу Администраторы.

Конец внимание 

Подготовка к подключению модуля

Перед подключением модуля в интерфейсе ELMA365 зайдите в свой аккаунт Jivo.

На боковой панели нажмите Управление > Каналы. Выберите настроенный канал, то есть сайт, на котором размещён виджет с чатом Jivo.

В левом меню выберите пункт Установка. В открывшемся окне скопируйте id установленного виджета.

Например, из строки <script src="//code.jivo.ru/widget/QaAFx87Ibf" async></script> нужно скопировать только QaAFx87Ibf.

Настройка модуля в ELMA365

Чтобы подключить модуль:

  1. Перейдите в раздел Администрирование > Модули.
  2. Нажмите кнопку +Модуль, а затем Скачать.
  3. В каталоге ELMA365 Store установите модуль Jivo.  
  4. После успешной установки нажмите Перейти в модуль.
  5. Нажмите Добавить элемент. Элемент представляет собой запись об одном подключённом онлайн-чате. Чтобы настроить интеграцию с несколькими настроенными чатами, добавьте в таблицу отдельный элемент для каждого из них.  

  1. В открывшемся окне заполните поля.

  • Название* — укажите название аккаунта;
  • Widget Id* — вставьте id виджета, скопированный в интерфейсе Jivo;
  • Приложение* — из выпадающего списка выберите Лиды или Сделки. После завершения диалога с клиентом в выбранном приложении будут создаваться новые элементы или дополняться имеющиеся;

После выбора приложения появятся дополнительные поля.

  • При завершении диалога всегда создавать новый элемент приложения — в поле доступно две опции:
    • Да — при каждом новом обращении в ELMA365 будет создаваться новый лид или сделка. История переписки отобразится в ленте элемента;
    • Нет — при новом обращении будет проверяться, есть ли в ELMA365 клиент с указанными контактными данными. Если запись о клиенте существует, диалог запишется в его ленту. Если записи нет, будет создана новая.
  • Воронка — выберите воронку, в которой будет создаваться новый элемент;
  • Поля блока Сопоставления — сопоставьте свойства из сервиса Jivo с контекстными переменными приложения из ELMA365. Это нужно для того, чтобы в новых элементах отображались данные пользователя, которые он указал в чате. Вы можете отредактировать или удалить добавленные по умолчанию поля, или создать новое с помощью кнопки +Добавить.

Обратите внимание, поля Email и Телефон посетителя нельзя редактировать или удалить, так как по ним проверяется наличие элемента в приложении или создаётся новый элемент;

  • Преднастроенные значения — нажмите кнопку +Добавить, чтобы задать для контекстных переменных выбранного приложения значения, которые нельзя получить из сервиса Jivo. Например, таким образом можно задать переменной значение, отвечающее за статус лида, чтобы новые лиды создавались уже со статусом Обработка, а не Новый.  
  1. Сохраните настройки.
  2. Скопируйте поле Webhook URL и нажмите Сохранить.

На этом настройка подключения модуля со стороны ELMA365 завершена. Далее необходимо настроить подключение со стороны Jivo.

Настройка модуля в Jivo

  1. В аккаунте Jivo перейдите в Управление > Каналы. Выберите настроенный канал и перейдите в Настройки интеграции для разработчиков.
  2. Включите Webhooks.
  3. В поле URL для Webhooks вставьте скопированный в интерфейсе ELMA365 на шаге 8 webhook URL.

После этого настройка интеграции модуля Jivo с ELMA365 будет завершена.

Обработка обращений в Jivo и передача данных в ELMA365  

Размещённый на сайте виджет с онлайн-чатом выглядит следующим образом:

Клиент отправляет запрос в чат, а оператор получает его и обрабатывает на сайте Jivo.

Чтобы информация о клиенте поступила в ELMA365, после беседы оператору необходимо завершить диалог. Для этого нужно нажать кнопку Решён, а затем Выйти из диалога.

Как только оператор завершит диалог в Jivo, в ELMA365 создастся новый лид или сделка по контактным данным, которые пользователь оставил в чате. В ленте на карточке элемента приложения отобразится переписка с оператором.

Если пользователь уже обращался, и по нему создан элемент приложения, история переписки отобразится в его ленте.
