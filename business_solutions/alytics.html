Модуль интеграции с Alytics

Модуль для интеграции с системой сквозной аналитики Alytics. Позволяет построить сквозную аналитику на основе данных о лидах и сделках из раздела CRM. С её помощью вы сможете отследить качество лидов, оценить окупаемость, а также проанализировать эффективность рекламных компаний.

начало внимание

Настраивать модули могут только пользователи, входящие в группу Администраторы.

конец внимание

Настроить модуль в ELMA365

Чтобы подключить модуль:

  1. Перейдите в раздел Администрирование > Модули > Alytics.
  2. Нажмите кнопку Включить.
  3. В появившемся окне Настройка интеграции укажите логин и пароль, которые будут использоваться в дальнейшем при добавлении проекта в системе Alytics.

  1. Создайте проект для отслеживания в системе Alytics. Для этого нажмите кнопку Добавить проект. В появившемся окне укажите название, например, Филиал Нижний Новгород, и нажмите Сохранить. В поле URL для интеграции сформируется ссылка, которую в дальнейшем необходимо использовать при добавлении проекта в системе Alytics.

  1. Нажмите кнопку Добавить цель, чтобы задать приложение и параметры, по которым будет построен отчёт в системе Alytics.
  2. В открывшемся окне настройте выборку данных:

  1. Заполните поля:
  • Название* — задайте название цели;
  • Приложение* — выберите приложение, по данным которого будет построен отчёт в системе Alytics;
  • Описание — добавьте описание цели.
  1. Настройте блок Фильтр — укажите, по каким параметрам элементы приложения будут передаваться в Alytics:
  • Связка — заполняется, если необходимо задать несколько условий, по которым будут отобраны элементы приложения. Можно выбрать один из двух вариантов:
    • И — отбираться будут элементы приложения, которые соответствуют каждому из заданных параметров;
    • ИЛИ — отобранные элементы будут соответствовать одному из заданных параметров.
  • Поле из приложения — поля приложения, по которым необходимо отобрать элементы;
  • Операция — оператор сопоставления для проверки каждого условия:
    • Знак = — элемент попадает в выборку, если поле приложения, заданное в столбце Поле из приложения, полностью соответствует значению из столбца Значение поля;
    • Знак ≠ — элемент попадает в выборку, если поле приложения, заданное в столбце Поле из приложения, не соответствует значению из столбца Значение поля;
  • Значение поля — значения для полей приложения из столбца Поле из приложения. В этом столбце вы можете указать конкретное значение полей или ввести значение вручную.
  1. Настройте блок Ответ — сопоставьте свойства из системы Alytics с полями приложения для выгрузки данных:
  • Свойства ответа — поля, предусмотренные в системе Alytics. В таблице приведены значения свойств ответа:

utm_source

рекламный источник

utm_medium

рекламный канал

utm_content

содержание рекламного объявления

utm_campaign

название и id кампании

utm_term

ключевое слово

client_id

id клиента

client_id_date

дата первой покупки клиента

number

номер транзакции

creation_date

дата и время создания элемента

revenue

выручка

profit

прибыль
  • Свойства приложения — поля приложения, которые будут передаваться в систему Alytics.
  1. Заполните поле URL цели, указав ID цели в Alytics. Обратите внимание, чтобы интеграция настроилась корректно, id целей в ELMA365 и в системе Alytics должны совпадать. Для первой цели в проекте используйте значение -1, для второй — 1. Id третьей и последующих целей указывайте по порядку. Ссылка в поле URL цели формируется автоматически.

Так может выглядеть окно добавления цели для выборки лидов, заинтересованных в конкретном продукте и квалифицированных с добавлением сделки:

  1. Сохраните настройки.

Для одного проекта можно добавлять несколько целей. Например, вы можете создать отдельные проекты для статистики по лидам и по сделкам в разных статусах.

При необходимости цель можно редактировать, нажав на значок карандаша напротив её названия, или удалить, нажав на значок корзины.

Настроить интеграцию в системе Alytics

После того как вы настроили модуль в ELMA365, необходимо добавить созданные проект и цели в систему Alytics.

Для этого:

  1. Зайдите в аккаунт Alytics.
  2. В верхнем меню перейдите в Настройки > Настройка целей.
  3. На открывшейся странице в блоке Данные из CRM нажмите на значок +.

Откроется окно подключения CRM.

  1. Перейдите в настройки модуля Alytics в ELMA365 и скопируйте URL проекта из поля URL для интеграции.
  2. В окне подключения CRM в Alytics вставьте скопированную ссылку в поле URL. Затем укажите логин и пароль, заданные в настройках интеграции в ELMA365, и нажмите кнопку Применить. Проект добавится в Alytics.
  3. В блоке Данные из CRM нажмите кнопку Создать цель. В открывшемся окне введите название цели и нажмите Подключить.

  1. После того как все цели добавлены, и интеграция настроена, обратитесь в поддержку Alytics по адресу support@alytics.ru для включения дополнительных настроек. В письме укажите номер проекта.

Если интеграция настроена верно, данные из ELMA365 появятся в проекте Alytics на следующий день после обращения на почту поддержи.

Более подробно об интеграции и работе с сервисом читайте в Инструкции Alytics.
