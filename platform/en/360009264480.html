Data export

You can export data from an app. For example, you can download a customer database, orders, invoices, or other documents. The data is saved to a .xlsx or .csv file, which allows you to later import data into other programs.

начало внимание

By default, all users can export data from apps. Learn how to restrict export in the Access to app data article.

конец внимание

To start exporting data, click the icon in the upper right corner of the app page. Select Data Export.

In the provided window, specify the following:

  • Format. You can save data either to an .xlsx or to a .csv file.
  • Export system information. Select this option to export system fields, codes, and identifiers in complex data types. It will allow saving links to related objects if in the future you plan to import data into another BRIX app. Suppose you need to export a list of customers. The app fields also contain links to contracts and agreements signed with customers. If you download the app with system information, after importing data into BRIX, all contracts and agreements will still be accessible from the app item page by clicking on the link.  
  • Export filtered data. Select this option to export data that you have previously selected by certain parameters using search or a filter. For example, it can be contracts filtered by signing date or contractor.
  • Export dates and convert to UTC+0. Select this option if you are uploading data with the Date/time type properties. It allows you to match the dates to the recipient company’s time zone.
  • Export the names of app items, users, files. Select this option if you want the linked objects’ data to be displayed as names without identifiers in the export file. To export both names and identifiers, check the Export system information box as well.

The data format used in the exported file is the same in data import.

начало внимание

You can download only the app data. Filters, images, files, tasks, comments, etc. will not be exported.

конец внимание

During export, all app fields are downloaded. If the app form has been configured in the advanced mode, the entire app context is exported.

If you do not check the options before starting the export, only identifiers will be uploaded for associated objects.

The file with the exported data will be saved in the Downloads folder on the hard drive of your computer.
