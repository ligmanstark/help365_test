Страница

Страница — это элемент левого меню, который отображает определенный набор данных при помощи виджетов. Вы можете создать страницу как внутри раздела, так и в левом меню главной страницы.

начало внимание

Создавать страницы и настраивать отображение виджетов на странице может только пользователь, включенный в группу Администраторы.

конец внимание

Добавить новую страницу

Для того чтобы добавить новую страницу:

  1. В меню раздела нажмите на кнопку Добавить.
  1. В открывшемся окне выберите блок Страницу.

  1. Заполните следующие поля:
  • Название страницы* — введите название страницы, которое будет отображаться в левом меню;
  • Ссылка на страницу* — ссылка формируется автоматически исходя из названия страницы, но на этапе создания ее можно изменить. Используется для быстрого доступа к странице;
  • Иконка страницы — выберите подходящую для страницы иконку. Иконки помогают пользователям визуально различать страницы в меню;
  • Количество колонок — опция позволяет расположить виджеты на странице в одну или в две колонки;
  • Видимость — укажите, кто сможет видеть название страницы в левом меню. Позднее эти настройки можно будет изменить:
    • Открыт всем — страница будет видна в меню всем пользователям системы;
    • Только администраторам — страница будет видна только пользователям, включенным в группу Администраторы.
  1. После того как поля будут заполнены, нажмите кнопку Создать.
  1. Откроется новая страница, которую нужно наполнить виджетами. Чтобы создать виджет на странице, нажмите на кнопку Добавить виджет.

Подробнее о настройке читайте в статье «Добавить виджет на страницу».

Управление страницей

Вы можете ограничить видимость страницы в левом меню, а также изменить ее настройки или удалить страницу из левого меню главной страницы или раздела.

Для этого нажмите на иконку справа от названия страницы и выберите соответствующий пункт меню.

Видимость страницы — выберите группы пользователей, которые должны видеть страницу в левом меню и нажмите на кнопку Сохранить.

Настройки страницы — вы можете изменить следующие поля: Название, Ссылка, Иконка, Количество колонок.

Копировать страницу — используйте это действие, если хотите создать новую страницу и продублировать для нее настройки текущей.

Удалить страницу — страница будет перемещена в Корзину, станет недоступна для всех пользователей и перестанет отображаться в левом меню. Удаленную страницу можно восстановить. Для этого зайдите в корзину и нажмите на кнопку Восстановить рядом с названием страницы.
