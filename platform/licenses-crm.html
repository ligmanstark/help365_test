Лицензирование ELMA365 CRM

Системное решение ELMA365 CRM включает в себя разделы, приложения, страницы, виджеты, бизнес-процессы и скрипты, которые позволяют выполнять задачи, связанные с продажами, маркетинговыми мероприятиями, ведением базы внешних контрагентов и взаимодействием с ними.

В системах до версии 2024.4 решение активировалось отдельно от платформы ELMA365 с помощью именных лицензий. Они привязывались к определённым пользователям, которым становились доступны все функциональные возможности ELMA365 CRM.

С версии системы 2024.4 ELMA365 CRM разделена на несколько решений, которые лицензируются по отдельности: ELMA365 Управление продажами и ELMA365 Маркетинг.

Кроме того, можно отдельно активировать  возможности ELMA365 CRM, не связанные с продажами напрямую. Для этого подключите дополнительные решения, чтобы вести клиентскую базу, коммуницировать с клиентами и создавать каталог продуктов.

Крупные компании с комплексным подходом к работе с клиентом могут объединить продажи, маркетинг и поддержку клиента в одно решение и подключить ELMA365 CX (Customer Experience).

Таким образом вы можете выбрать подходящий способ лицензирования ELMA365 CRM в зависимости от используемой поставки ELMA365 и необходимых функциональных возможностей CRM.

Рассмотрим доступные cпособы подключения функциональных возможностей ELMA365 CRM.

ELMA365 CX (Customer Experience)

Комплексное решение для взаимодействия с клиентами, предоставления им продуктов и услуг, а также формирования высокой степени их удовлетворённости сервисом.

Решение предназначено для поставок ELMA365 SaaS Enterprise и On-Premises Enterprise. С его помощью можно автоматизировать процессы продаж от первого получения лида до послепродажного сервисного обслуживания клиента.

ELMA365 CX включает в себя:

  • основной компонент Платформа ELMA365;
  • решение ELMA365 Service;
  • решение ELMA365 Управление продажами;
  • решение ELMA365 Маркетинг.

Для подключения выбирается количество пользователей, которые будут работать с решением, и активируются именные или конкурентные лицензии. Чтобы определить сотрудников с именными лицензиями, используйте группу Привилегированные пользователи.  

ELMA365 Управление продажами

Решение предоставляет большинство функциональных возможностей ELMA365 CRM и используется для организации и отслеживания работы с клиентами, структурирования клиентской базы, учёта финансовых ресурсов и прогнозирования доходов компании.

Решение активируется отдельно от платформы ELMA365 и подходит для любых её поставок. Для подключения используются именные и конкурентные лицензии. Чтобы определить сотрудников с именными лицензиями, используйте группу Пользователи решения Управление продажами.

В решение входят компоненты и функции для:

  • поддержания коммуникаций с клиентом — встроенный почтовик, телефония, веб-формы, хранение истории активностей, определение стадии прогрева. Эти функции CRM включены в решение ELMA365 Управление коммуникациями;
  • ведения клиентской базы — поиск и обработка дублей в приложениях CRM по преднастроенным правилам и в других приложениях системы, обработчик событий для выявленных дублей, виджет Иерархия. Эти функции CRM выделены в решение ELMA365 Управление контрагентами;
  • создания каталога простых и конфигурируемых продуктов — группировка продуктов и их характеристик, настройка правил заполнения характеристик при выборе продукта для клиента, виджет Выбор продукта. Эти функции CRM выделены в решение ELMA365 Продуктовый каталог;
  • внедрения процессов продаж:
    • лиды и сделки;
    • преднастроенные задачи CRM и создание пользовательских типов задач;
    • назначение задач CRM на пользователей вручную и в рамках бизнес-процессов с помощью специальных графических элементов;
    • планирование и прогнозирование поступлений, возможность задавать и отслеживать цели продаж в компании.

ELMA365 Маркетинг

Решение используется для формирования сегментов клиентов, отправки им массовых рассылок писем и осуществления долгосрочной поддержки.

Активируется отдельно от платформы ELMA365 с помощью дополнительных именных лицензий.

Чтобы предоставить пользователям доступ к решению, используйте группу Пользователи решения Маркетинг.

Решение включает в себя раздел Маркетинг и его возможности:

  • приложение Подписчики и выявление записей-дублей в базе контактов по электронной почте;
  • создание статистических и динамических сегментов подписчиков с помощью: настройки параметров фильтрации, EQL-запросов и методов API;
  • создание и отправка массовых рассылок и специальные методы API для работы с ними;
  • отображение рассылок по клиенту в специальном виджете;    
  • учёт маркетинговых мероприятий в отдельном справочнике;
  • карты коммуникаций для отправки триггерных рассылок сегментам подписчиков. Доступны только в поставке On-Premises;
  • графический редактор для создания шаблонов писем.

Функции CRM для расширения возможностей других решений

В ELMA365 предусмотрено лицензирование части функций CRM без компонентов, отвечающих за ведение продаж.

Для этого используются специальные решения ELMA365 Управление коммуникациями, ELMA365 Управление контрагентами и ELMA365 Продуктовый каталог. Подключить их можно, например, если продажи осуществляются в другой системе.

Для активации каждого решения используется серверная лицензия. Она активируется дополнительно к лицензии Платформа ELMA365, подходит для любых её поставок и обеспечивает доступ к подключаемым функциям в рамках одной компании.

Рассмотрим подробнее дополнительны решения, подключаемые функций ELMA365 CRM:

  1. ELMA365 Управление коммуникациями — используется для поддержания и отслеживания коммуникаций с клиентами. С лицензией активируются возможности:
    • использование встроенного почтового клиента и отправка писем в ходе бизнес-процессов;
    • подключение провайдеров телефонии;
    • создание веб-форм;
    • создание пользовательских активностей менеджеров по взаимодействию с клиентом и отслеживание истории активностей;
    • присвоение клиентам стадий прогрева.
  1. ELMA365 Управление контрагентами — применяется для организации работы с клиентской базой и включает следующие возможности:
    • поиск и обработка записей с одинаковыми данными в разделе CRM по преднастроенным правилам;
    • организация поиска дублей в других приложениях системы;
    • создание обработчика события Дубли при выявлении элементов-дублей;
    • виджет Иерархия для отображения иерархической структуры компаний-контрагентов.
  1. ELMA365 Продуктовый каталог — применяется для создания иерархического каталога простых и конфигурируемых продуктов компании. Включает следующие возможности:
    • группировка продуктов и их характеристик по тематике;
    • настройка правил заполнения конфигурируемых характеристик продукта;
    • выбор продукта на карточке элемента приложения с помощью виджета и заполнение характеристик продукта в соответствии с настроенными правилами.

Устаревшая поставка ELMA365 CRM

ELMA365 CRM — устаревший вариант поставки решения CRM, подключённый до релиза системы 2024.4. Активируется с помощью именных лицензий. Если в компании используется этот вид лицензирования, вы можете продлить срок его действия. Для новых клиентов эта лицензия не доступна.

Включает в себя:

  • решение ELMA365 Управление продажами;
  • решение ELMA365 Маркетинг без доступа к картам коммуникаций и редактору шаблонов.

Продление активации для ELMA365 CRM и покупка дополнительных лицензий

 
Активацию устаревшей поставки ELMA 365 CRM можно продлить. Также доступна покупка дополнительных именных лицензий аналогичного типа — ELMA365 CRM.

Одна именная лицензия ELMA365 CRM включает в себя функциональные возможности, предоставляемые для одной лицензии ELMA365 Управление продажами и одной лицензии ELMA365 Маркетинг.

 
Чтобы продлить поставку или приобрести лицензии, обратитесь к менеджеру ELMA365.

 
Обратите внимание, нельзя активировать и использовать вместе устаревшую поставку ELMA365 CRM и решения нового формата: ELMA365 Управление продажами, ELMA365 Маркетинг, ELMA365 Управление коммуникациями, ELMA365 Управление контрагентами и ELMA365 Продуктовый каталог.

 
Компания может перейти на новый формат лицензирования CRM, чтобы реализованные функциональные возможности обновлялись, согласно выпускаемым версиям платформы. Чтобы выбрать и активировать новый тип лицензирования, отвечающий потребностям компании, обратитесь к менеджеру ELMA365.

Активировать решение для SaaS

Демоверсия SaaS

Со всеми возможностями CRM можно ознакомиться бесплатно, используя его демоверсию и любое количество лицензий. Срок активации демоверсии ограничен и согласовывается с менеджером ELMA365.

Активировать демоверсию или решение может администратор системы. Для этого:

  1. Перейдите в раздел Администрирование > Управление лицензиями.
  2. В карточке нужного решения, нажмите Связаться с менеджером и отправьте запрос. Если вы активируете именную и конкурентную лицензию, определите, сколько пользователей должны получить доступ к решению.
  3. Менеджер ELMA365 согласует с вами количество именных и конкурентных лицензий для решений, которым они необходимы, и срок активации демоверсии. Затем менеджер ELMA365 активирует решение.
  4. В карточке решения вы увидите статус Активирован, срок действия демоверсии и количество именных и конкурентных лицензий.
  5. Привяжите именные лицензии к пользователям, добавив их в соответствующую группу.

Активировать платное решение в SaaS

Чтобы продолжить работу с решением CRM после пробного периода:

  1. В разделе Администрирование > Управление лицензиями выберите решение и нажмите Связаться с менеджером.
  2. После оплаты лицензий менеджер ELMA365 активирует решение. Его статус изменится на Активирован. Рядом вы увидите срок действия лицензий, совпадающий со сроком действия вашей подписки, а также количество лицензий.

Если вы хотите купить дополнительные лицензии решения, обратитесь к менеджеру ELMA365, у которого приобрели лицензии ранее. После покупки лицензий добавьте новых пользователей в группы.

Активировать решение для On-Premises

Демоверсия On-Premises

Со всеми возможностями CRM можно ознакомиться бесплатно, используя его демоверсию и любое количество лицензий. Срок активации демоверсии ограничен и согласовывается с менеджером ELMA365.

Активировать демоверсию или решение может администратор системы. Для этого:

  1. Перейдите в раздел Администрирование > Управление лицензиями.
  2. В карточке решения нажмите Активировать.
  3. В открывшемся окне скопируйте Регистрационный ключ сервера и передайте его менеджеру ELMA365, у которого была куплена платформа. Если для работы с решением нужны именные или конкурентные лицензии, заранее определите их количество.
  4. Менеджер ELMA365 согласует с вами срок действия демоверсии и сгенерирует ключ активации решения. Вставьте его в поле Ключ активации и нажмите Активировать.

  1. После этого статус решения изменится на Активирован. Рядом вы увидите срок действия демоверсии и количество лицензий.
  2. Привяжите именные лицензии к пользователям, добавив их в соответствующую группу.

Активировать платное решение в On-Premises

Чтобы продолжить работу с решением CRM после пробного периода:

  1. В разделе Администрирование > Управление лицензиями выберите решение и нажмите Активировать.

Если вы используете демоверсию и срок её действия не истёк, нажмите Обновить лицензию. В появившемся окне скопируйте Регистрационный ключ сервера и передайте его менеджеру ELMA365. Согласуйте количество именных или конкурентных лицензий и их стоимость.

  1. После оплаты лицензий менеджер ELMA365 сгенерирует ключ активации решения. Вставьте его в поле Ключ активации и нажмите кнопку Активировать. Срок активации не ограничен.

Если вы хотите купить дополнительные лицензии, обратитесь к менеджеру ELMA365, у которого приобрели лицензии ранее. После покупки лицензий менеджер ELMA365 сгенерирует ключ активации.

Чтобы активировать решение с новыми данными о его лицензиях:

  1. В разделе Администрирование > Управление лицензиями выберите решение и нажмите Обновить лицензию.
  2. Вставьте новый ключ активации и нажмите кнопку Активировать.
  3. Добавьте новых пользователей в соответствующую группу.

Привязать лицензии к пользователям

Для лицензий с типом Именная определите, кто из пользователей сможет всегда иметь доступ к активированным функциям CRM. Для этого добавьте пользователей в соответствующие группы:

  1. Перейдите в раздел Администрирование > Группы и откройте папку Системные группы.
  2. Выберите группу:
  • для ELMA365 Управление продажами и устаревшей поставки ELMA365 CRM — группа Пользователи решения Управление продажами;
  • для ELMA365 Маркетинг — группа Пользователи решения Маркетинг;
  • для ELMA365 CX, ELMA365 Управление коммуникациями, ELMA365 Управление контрагентами и ELMA365 Продуктовый каталог — группа Привилегированные пользователи;
  1. В открывшемся окне нажмите Редактировать.
  2. Укажите участников группы. Для выбора доступны все пользователи системы. Обратите внимание, их количество не может превышать количества именных лицензий активированного решения.

Пользователи без лицензии не увидят в системе лицензируемых компонентов CRM. Для такого пользователя отобразятся только приложения Компании и Контакты в разделе CRM.

В операциях и процессах с участием пользователя без лицензии возникнут ошибки. Например, если в карточке сделки написать сообщение пользователю, у которого нет лицензии, отобразится предупреждение. Такой пользователь не получит отправленные ему сообщения.

После привязки лицензии к пользователю можно возобновить запущенные экземпляры процессов, при исполнении которых возникла ошибка по причине отсутствия лицензии.
