Архитектура и системные требования / Рабочее место

Рабочее место

Система ELMA365 имеет клиент-серверную модель взаимодействия. Пользователи взаимодействуют с сервером через сетевое соединение-локальную сеть или интернет. Для подключения к серверу ELMA365 не требуется специальное клиентское программное обеспечение, пользователи работают с системой через веб-браузер. Дополнительно для работы пользователей в системе может использоваться классическое Windows/macOS приложение, которое позволяет получать оповещения с закрытым браузером. Также можно установить мобильное приложение ELMA365.

Поддерживаемые браузеры

Для всех редакций в поставках SaaS и On-Premises поддерживаются следующие браузеры (информация дана с указанием минимальных требований к версиям браузеров для ELMA365, начиная с релиза 2025.1):

  • Google Chrome — не ниже версии 110;
  • Microsoft Edge — не ниже версии 110;
  • Firefox — не ниже версии 110. Обратите внимание, работа в режиме инкогнито в браузере Firefox невозможна, так как в этом режиме Firefox не поддерживает IndexedDB;
  • Opera — не ниже версии 96;
  • Yandex Browser — не ниже версии 23;
  • Safari для macOS и iOS — не ниже версии 15;
  • Браузер Android — две последние версии.

Если ваша версия ELMA365 ниже 2025.1, для корректной работы системы рекомендуем использовать версию браузера, выпущенную за последние два года.

Системные требования для ПК при работе с ELMA365 в браузере соответствуют требованиям, заявленным для корректной работы самого браузера.

Также для работы в браузере рекомендуем использовать актуальную операционную систему, обновляемую производителем.

Приложение ELMA365 для ПК

При желании в ELMA365 можно работать через приложение для персональных компьютеров. Требования для корректной работы приводятся ниже:

Операционная система

OC Windows (не ниже версии 10), Linux with Desktop Environment, macOS

Минимальные аппаратные требования

Процессор: Тактовая частота 1 ГГц и выше; количество ядер 1 и выше

Жёсткий диск: 200 МБ (приложение) + ~1ГБ (кеш)

Оперативная память: ~500 МБ

Разрешение экрана: 1024x768 и выше, цвет 16 бит и выше

Сетевое подключение: канал связи 256 Кбит/сек и выше

Мобильное приложение ELMA365

Вы можете использовать для работы мобильное приложение ELMA365. Требования для его установки из AppStore или GooglePlay и подключения:

Операционная система
  • iOS (iPad, iPhone) — не ниже версии 15;
  • Android с поддержкой Google Play services — не ниже версии 6

Минимальные аппаратные требования

Жёсткий диск: 200 МБ (приложение) + ~1ГБ (кеш)

Свободная оперативная память: 500 МБ

Сетевое подключение: канал связи 256 Кбит/сек и выше

Требования для подключения

Минимальные версии браузеров для версий ELMA365, выпущенных в 2025 году:

  • для iOS — не ниже версии 16 (версия Safari / WebView соответствует версии iOS);
  • для Android — не ниже версии 6 + WebView не ниже версии 106;
  • для Google Chrome — не ниже версии 110.
