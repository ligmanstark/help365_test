Массовые действия с элементами приложений

После того как администратор включил и настроил массовые действия, пользователи смогут применять их на странице приложения.

Например, удалять все ненужные договоры, изменять дату поставки нескольких заказов или выставлять новый статус для выбранных сделок. Кроме того, можно запустить бизнес-процесс с группой элементов, например, назначить ответственного за все поступившие заказы.

начало внимание

Использовать массовые действия можно только при режиме отображения элементов в виде таблицы.

конец внимание

Применить массовые действия

Для использования массового действия:

  1. В правом верхнем углу страницы приложения выберите табличный режим отображения .
  2. Отметьте несколько элементов.
  3. Чтобы выбрать все видимые на странице элементы, поставьте флажок в строке Название. Для применения действия ко всем созданным в приложении записям, в появившемся окне нажмите Выбрать все элементы и подтвердите действие.

  1. Нажмите на появившуюся в правом верхнем углу кнопку С выделенными.
  2. Выберите из списка одно из доступных массовых действий: Редактировать, Удалить, Восстановить, Сменить статус, Запустить бизнес-процесс.

Массовое редактирование

При выборе действия Редактировать откроется настроенная администратором форма массового редактирования. Поля на ней будут пустыми. Вы сможете ввести в них новые значения, которые применятся ко всем редактируемым элементам.

На правой панели формы отображается список всех выбранных элементов. Нажав на значок корзины рядом с названием элемента, его можно удалить из списка редактирования.

После заполнения всех полей нажмите Сохранить.

Массовое удаление / восстановление

Выделите все элементы, которые нужно удалить и нажмите С выделенными > Удалить.

Удаленные элементы можно затем массово восстановить. Для этого откройте поиск по параметрам и в правом верхнем углу нажмите на Удаленные. Отметьте те элементы, которые хотите восстановить и нажмите на кнопку С выделенными > Восстановить.

Массовая смена статуса

Если в приложении настроены статусы и включена их ручная смена, становится доступной массовая смена статуса.

При выборе этой опции откроется окно с тремя вкладками:

  • Все элементы — эта вкладка будет активной, если возможность сменить статус есть для всех выбранных элементов. Например, если для каждого статуса настроен переход только вперед и у всех выбранных элементов есть следующий статус, то их можно будет все перевести в следующий статус, выбрав его в выпадающем списке. В других случаях эта вкладка будет неактивна и по умолчанию будет открыта вкладка По статусам.
  • По статусам — здесь элементы будут сгруппированы по статусам. Для каждой группы элементов можно выбрать новый статус из списка.
  • Каждый элемент — на этой вкладке вы сможете изменить статус у каждого выбранного элемента в отдельности.

После того, как вы указали новые статусы, нажмите Сохранить.

Массовый запуск процесса

Для запуска экземпляра процесса по отмеченным элементам нажмите кнопку С выделенными и выберите настроенную администратором кнопку.

Обратите внимание, если внутри бизнес-процесса не настроен цикл выполнения, со всеми выбранными элементами запускается один экземпляр.
