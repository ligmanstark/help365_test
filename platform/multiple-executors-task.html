Задачи с несколькими исполнителями

При создании задачи вы можете указать больше одного исполнителя — нескольких пользователей, группу, отдел и т. д. В этом случае на каждого сотрудника ставится отдельный экземпляр задачи, а автор может увидеть результаты работ в одном окне. Это удобно, когда нужно поставить однотипную задачу на группу сотрудников, например, попросить всех руководителей подготовить отчёт.

У задач с несколькими исполнителями есть особенности:

  • автор задачи может отслеживать прогресс всех сотрудников, а также видеть результаты работы каждого в сводной таблице;
  • изменить общее количество исполнителей нельзя, но можно переназначить экземпляр, то есть заменить одного исполнителя на любого другого пользователя;
  • автор может изменить детали задачи, и данные обновятся в карточках всех исполнителей;
  • при отмене задачи автором исполнение прерывается для всех сотрудников;
  • лента задачи является общей для всех экземпляров. В ней могут вести переписку автор, согласующий, исполнители и участники задачи;
  • при работе над задачей сотрудник видит карточку своего экземпляра, где в качестве исполнителя указан только он;
  • при завершении задачи с несколькими исполнителями сотрудник может добавить вложение.

Отслеживание прогресса исполнителей

При назначении задачи на нескольких сотрудников автор может просматривать статусы исполнения и результаты работы каждого из них в сводной таблице.

Для этого перейдите в раздел Задачи > Исходящие. В списке отобразятся названия всех незавершённых экземпляров задачи. Откройте любой из них. Под информацией о задаче вы увидите таблицу Результаты.

Здесь отображаются все сотрудники, которые работают над задачей, и статусы исполнения. Чтобы обновить данные, в правом верхнем углу таблицы нажмите Обновить.

Когда сотрудник завершает задачу, в таблице отображается его комментарий и прикреплённое вложение.

Для работы с вложением наведите курсор на значок скрепки. Вы можете:

  • — загрузить файл на компьютер;
  • — выбрать папку из раздела Файлы и сохранить в ней копию вложения;
  • — скопировать ссылку на загрузку файла;
  • — открыть файл в режиме просмотра.

На боковой панели карточки отображается общий статус задачи. Так, пока все сотрудники не завершат работу, задача остаётся в статусе На исполнении. Если кто‑то из сотрудников отметит задачу как невозможную для выполнения, после завершения работ задача приобретёт статус Частично выполнена. Подробнее о статусах читайте в статье «Карточка задачи».

После того, как все исполнители завершат задачу, перейти к сводной таблице можно из раздела Задачи > Исходящие > Все. Для этого нажмите на перечёркнутое название любого экземпляра задачи.

Действия с задачей

Пользователи, которые работают над задачей, могут выполнять определённые действия с ней: переназначать, изменять список участников, редактировать информацию и т. д.

Подробнее о доступных действиях читайте в статье «Действия с задачей».

Отмена задачи

Автор задачи может прервать её исполнение. Для этого нужно открыть её карточку и на нижней панели нажать Прервать процесс.

Обратите внимание, отменяются все экземпляры задачи. Невозможно отменить задачу только для одного исполнителя.

Переписка по задаче

Для автора, согласующего, исполнителей и участников задачи доступна единая лента. Любой из этих сотрудников может просматривать всю переписку и отвечать на сообщения. Все пользователи, задействованные в работе, увидят оповещения о новых сообщениях в своей #ленте.

Работа с задачей

Если на вас назначена задача, в которой указано несколько исполнителей, она появится в разделе Задачи > Входящие. Её карточка выглядит так же, как у стандартной задачи. В ней не отображается полный список исполнителей. Вы увидите необходимую для работы информацию: автора, срок, описание, приоритет, прикреплённые файлы.

В нижней части карточки отображается лента задачи. Здесь можно обсудить вопросы с коллегами или уточнить информацию у автора. Все сотрудники, которые задействованы в выполнении задачи, в том числе другие исполнители, увидят уведомления о новых сообщениях и смогут участвовать в переписке.

Завершение задачи

После выполнения работы откройте карточку задачи и на нижней панели нажмите кнопку Сделано.

Для задачи с несколькими исполнителями доступно прикрепление вложения. Его увидит только автор задачи в таблице результатов.

Чтобы загрузить файл с компьютера, нажмите + Файл. Для выбора из раздела Файлы нажмите на три точки в правой части поля.

Если по какой-то причине вы не можете завершить задачу, нажмите кнопку Невозможно выполнить. В этом случае вы также можете прикрепить файл.

Автор задачи увидит в таблице результатов новый статус исполнения, а также ваш комментарий и вложение, и сможет отреагировать на ситуацию.
