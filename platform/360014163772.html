Поиск по задачам

Чтобы быстро найти нужную информацию о задаче, воспользуйтесь поиском. Вы можете искать задачи по названию, автору, дате создания и завершения, а также по полям связанного с задачей бизнес-процесса или приложения.

Если вы планируете использовать несколько раз один и тот же набор параметров для поиска задач, вы можете сохранить его как фильтр.

Поиск по названию

Чтобы найти задачу, просто введите её название в строке поиска. В зависимости от того, на какой странице вы находитесь, поиск будет выполняться по типам задач: Входящие, Исходящие, Задачи CRM, Участвую и т. д.

В результатах поиска, помимо названия, показывается автор задачи и сроки её выполнения.

Вы можете отфильтровать результаты поиска. Для этого переключайтесь между вкладками Просроченные (незавершённе задачи в группе Задачи CRM, у которых истёк срок выполнения), Текущие (актуальные задачи), Завершенные (выполненные задачи в группе Задачи CRM), Все (текущие и завершённые задачи).

Поиск по параметрам

Чтобы результаты поиска были более точными, используйте расширенный поиск. Для этого нажмите на значок в строке поиска.

Вы можете искать задачу:  

  • по основным полям — названию, сроку выполнения, исполнителю для задач CRM;  
  • по системным полям — автору, дате создания;
  • по полям связанного объекта — объектом может быть бизнес-процесс, в ходе которого выполняется задача, или приложение, с элементом которого она связана. Связь задачи с приложением осуществляется одним из следующих способов:
    • в контекстную переменную бизнес-процесса передаётся элемент приложения через опцию Связать с полем при запуске бизнес-процесса с карточки элемента приложения или через блок Запуск процесса;  
    • бизнес-процесс запускается с элементом связанного с ним приложения;
    • задача создаётся на карточке элемента приложения с помощью виджета Связанные задачи.  

Введите название связанного объекта или выберите его из существующих, нажав на значок лупы. Указать можно системные и пользовательские объекты, для бизнес-процесса вы также можете выбрать названия задач CRM. После этого в окне поиска появятся поля связанного объекта, для которых включена опция поиска и сортировки. С их помощью вы можете ограничить критерии поиска задачи, например, указав определённую дату или участника.

Использование фильтров

Каждый пользователь может задать набор параметров для поиска задач и сохранить их как фильтр, чтобы использовать такой поиск многократно.

Создать фильтр

Чтобы добавить фильтр, откройте окно поиска, укажите необходимые параметры и нажмите кнопку Сохранить как фильтр.

В появившемся поле введите название фильтра и нажмите Сохранить.

Применить фильтр

Добавленные фильтры будут отображаться на боковой панели окна расширенного поиска.

Чтобы применить фильтр, нажмите на его название. В списке задач отобразятся записи, которые удовлетворяют параметрам поиска.

Кроме того, если вы создали фильтры для страницы Входящие, их названия будут отображаться в левом меню раздела. Вы сможете быстро сортировать задачи по группам, перемещаясь между страницами.

Отредактировать фильтр

Чтобы удалить, переименовать фильтры или изменить порядок их отображения, откройте окно расширенного поиска и в правом верхнем углу нажмите на значок карандаша.

Введите новое название фильтра или удалите его, нажав на значок корзины. Для перемещения зажмите значок и перетащите фильтр.

Чтобы выйти из режима редактирования, нажмите на значок .

Внесённые изменения отобразятся в окне расширенного поиска и в левом меню раздела для фильтров страницы Входящие.
